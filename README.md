# Quiz Islam (Web) — 200 questions

Web app de quiz Islam prête à publier sur GitHub Pages.

## Fonctionnalités
- 200 questions (JSON)
- 4 choix, 1 bonne réponse
- Timer 10 secondes
- Mélange des questions
- Score + explication courte après chaque réponse
- Responsive (mobile/desktop)

## Lancer en local
Ouvre `index.html` (ou utilise un serveur local).

## Publier sur GitHub Pages
1) Push ce repo sur GitHub  
2) Settings → Pages → Deploy from branch → `main` / `/root`  
3) Ton quiz est en ligne

## Données
Fichier: `assets/quiz_islam_200.json`

## Sons (optionnel)
Place tes fichiers dans `assets/sounds/` puis modifie `SOUNDS` dans `app.js`.
